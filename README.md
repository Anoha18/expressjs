# Пример работы express js
## Скачиваем либо клонируем:
`` `
https://gitlab.com/AnokhinVL/expressjs.git
`` `

## Устанавливаем зависимости
`` `
npm install
`` `

## Запускаем
`` `
npm run start
`` `

## Построен с 
* [Express](https://expressjs.com/ru/) - серверный фраемворк
* [Vue](https://vuejs.org/) - клиентская часть
